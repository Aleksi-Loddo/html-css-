class and id in html

you can use one id only once in your html page, but you can use one class several times in your html page

in css you have to use symbol # to choose an id. and you have to use symbol . to choose a class.

div and span

you can use div and span to mark some sections of your html, in order to format it with css

the difference between them is that div is a block tag and span is an inline tag.
